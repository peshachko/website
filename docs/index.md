# Добре дошли

Ние сме група ентусиасти, които силно вярваме, че движението е в основата на едно
усмихнато ежедневие. То води, някак естествено, към по-отворена комуникация и динамичен
начин на живот. Това е нашата страница -- тук ще намерите информация за минали и
планирани събития, както и кратки статии на теми които са ни мотивирали по някакъв
начин.
